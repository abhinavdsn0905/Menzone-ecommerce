# 🛍️ MENZONE – Men’s Wear E-Commerce Website

MENZONE is a full-stack **Django-based e-commerce web application** developed as a college project.  
It provides a complete shopping experience for men’s fashion, including product browsing, cart management, order placement, and an admin dashboard.

---

## 🚀 Features

### 👤 User Features
- User registration & login
- Browse men’s fashion collections
- Category-wise product listing
- Add products to cart
- View and manage cart
- Checkout & place orders
- Order confirmation email after checkout
- View user order history
- Modern, responsive UI

---

### 🔐 Admin Features
- Separate admin login (custom admin panel)
- Admin dashboard with statistics
- View total users, products, and orders
- Add, edit, and delete products
- View all user orders
- Delete users
- Clean and organized admin UI

---

## 🛠️ Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Python, Django
- **Database**: SQLite
- **Authentication**: Django Authentication System
- **Email**: SMTP (Gmail)
- **Version Control**: Git & GitHub
