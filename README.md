# 🛍️ MENZONE – Men’s Fashion E-Commerce Website

MENZONE is a full-stack **Django-based e-commerce web application** focused on men’s fashion.  
It includes **user authentication with OTP**, **shopping cart**, **order management**, and a **custom admin panel** for managing products, users, and orders.

---

## 🚀 Features

### 👤 User Features
- User registration & login with **Email OTP authentication**
- Browse products by categories:
  - Shirts
  - T-Shirts
  - Jeans
  - Jackets
- Add products to cart
- Remove products from cart
- View cart summary (total items & price)
- Checkout and place orders
- Receive **order confirmation email**
- View previous orders in **My Orders** page

---

### 🛒 Cart & Orders
- Cart linked to logged-in user
- Quantity management
- Secure checkout (login required)
- Orders saved with status and total amount
- Order history per user

---

### 🛠️ Admin Panel (Custom)
- Admin login (separate from user login)
- Dashboard with:
  - Total Products
  - Total Orders
  - Total Users
- Add / Edit / Delete products
- View all orders
- Delete registered users
- Clean sidebar-based admin UI

---

## 🧑‍💻 Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS
- **Database:** MySQL
- **Authentication:** Django Auth + Email OTP
- **Email Service:** SMTP (Gmail)
- **Version Control:** Git & GitHub
